## Overview

Safar Logistics processes 90,000–150,000 truck dispatches daily. Before every dispatch, a critical safety checkpoint verifies that each truck is properly loaded and secured. This process was digitized in 2018 via an internal tool called **LoadX** — but the system still relied heavily on manual effort across job assignment, image review, audit sampling, KPI reporting, and feedback loops.

This case study covers the full business analysis for upgrading LoadX into a structured, automated, and data-driven platform.

## Projected Impact

| Metric | As-Is | To-Be |
|---|---|---|
| Associate Throughput | 300 jobs/day | 840 jobs/day (3× increase) |
| Accuracy | ~90% | 97% |
| Average TAT | ~84 seconds | ~30 seconds |
| Feedback Loop | 1–2 days | Under 5 minutes |
| Audit Coverage | 5–10% | 10–15% |
| Reporting | Manual (Excel) | Real-time dashboards |
